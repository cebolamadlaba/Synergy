Add your sql migration scripts following this file name structure. Example: V1_3_1__Create_table.sql:
prefix: configurable, default: V
version: numbers separated by _ (one underscore)
separator: configurable, default: __ (two underscores)
description: words separated by underscores
suffix: configurable, default: .sql

For more info check :https://github.com/lecaillon/Evolve
